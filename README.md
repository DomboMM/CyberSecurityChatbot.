# 📘 Cybersecurity Awareness Chatbot - Part 1

Welcome to the Cybersecurity Awareness Chatbot! This program educates users about online safety, including password security, phishing, and safe browsing.

---

## 🛠️ Features Implemented

1. **Voice Greeting**  
   - Plays a recorded welcome message (`greeting.wav`) when the application starts.

2. **ASCII Logo / Image Display**  
   - Displays a cybersecurity-themed ASCII logo at launch.

3. **Personalized User Interaction**  
   - Asks for the user's name and addresses them directly.

4. **Basic Response System**  
   - Responds to questions about:
     - Password safety  
     - Phishing  
     - Safe browsing  
     - Chatbot purpose  
   - Default response for unknown inputs: "I didn't quite understand that. Could you rephrase?"

5. **Input Validation**  
   - Handles empty or invalid inputs gracefully.

6. **Enhanced Console UI**  
   - Uses colors, spacing, and borders for readability.

7. **Modular Code Structure**  
   - `Program.cs` controls flow  
   - `Chatbot.cs` contains the main chatbot logic  
   - `AudioPlayer.cs` handles audio  
   - `ConsoleUI.cs` handles ASCII logo and console formatting
